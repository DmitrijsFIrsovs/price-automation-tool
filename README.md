# 🛒 Price Automation Tool

Python application for automating price synchronization between Google Sheets and an e-commerce administration panel.

## Features

- Reads product prices from Google Sheets
- Supports multiple marketplaces (RD, 1A, Varle, Shop)
- Updates only changed prices
- User-friendly desktop interface built with Tkinter
- Automates browser interactions with Selenium
- Logs progress and errors

## Technologies

- Python
- Selenium
- Tkinter
- Google Sheets API
- OAuth2

## Business Value

This tool automates repetitive price update tasks, reducing manual work and minimizing pricing errors across multiple sales channels.

## Installation

```bash
pip install -r requirements.txt
python price_automation_tool.py
```
